软件导航

一、软件资讯

▼ 国内网站 简介
🌐 GitHub 全球最大开源托管平台，开发者们“殿堂般”。
🌐 52Pojie GitHub 和 52Pojie 合称“北乔峰南慕容”，国内活跃度最高的软件交流社区。
🌐 小众软件 老牌软件分享网站，介绍各种新鲜软件，还新开了一个“发现分享”频道。
🌐 异次元软件 与“小众软件”类似，但更新强度大不如以前。
🌐 Topbook 可以说是“视频版小众软件”，信噪音比不高，同样的内容需要的时间更多。
🌐 少数派 软件界的“小红书”，文章更多的是“经验分享”，但苹果用户占比太高。
🌐 IT 之家 专业的 IT 新闻网站，有一个“软件”频道，软件界的大新闻，这里都有。
▼ 国外网站 简介
🌐 Product Hunt 国外最火的软件应用资讯网站，能发掘很多前沿新产品。
🌐 BetaList 与 Product Hunt 类似，但提供的都是新鲜前沿的应用。
🌐 Slant 提供各种应用的排名测评，有正反观点，比如“最好的浏览器是什么”？
▼ 相似软件 简介
🌐 Product Hunt 网外最火的软件资讯网站，右下角的”View all alternatives”就是相似软件。
🌐 AlternativeTo 老牌网站，支持 Windows、macOS、linux、网页、iOS、安卓。
🌐 Altapps.net 支持多语言，支持多平台，资源也比较丰富。
🌐 ProgSoft 支持桌面软件，热度也很高。
🌐 Alternative.me 支持桌面软件，资讯相对较老。
🌐 Alternatives.app 支持桌面软件，资讯相对较老。
🌐 Similarsites 发明相似网站。
🌐 Similarweb 发明相似网站和手机应用。
🌐 开源替代方案 200 多个开源替代品，可替代企业日常运营所需的工具。
🌐 MrFreeTools 查找收费应用的免费方案。

二、软件论坛

▼ 国内论坛 简介
🌐 吾爱破解 国内活跃度最高的软件交流社区。
🌐 V2EX 前知名开发者交流社区，现在需要“魔法”才能访问了。
🌐 远景论坛 国内知名的 Windows 软件交流社区。
🌐 卡饭论坛 一个以“卡巴斯基”为起点的软件交流社区，需要登录才能访问。
🌐 虫部落 一个以搜索主题的资源交流社区。
🌐 飘云阁 类似 52Pojie 的论坛。
🌐 无忧启动 类似 52Pojie 的论坛。
🌐 吾爱汇编 类似 52Pojie 的论坛。
🌐 批处理之家 一个专注于 .bat 批处理文件的软件论坛。
▼ 国外论坛 简介
🌐 RuTracker.org 提供 BT 下载，综合性资源下载网站，各种内容都有，俄罗斯知名网站。
🌐 Mobilism 提供 BT 下载，综合性资源下载网站，各种内容都有。
🌐 Nsane Forums 提供 BT 下载，综合性资源下载网站，各种内容都有。

🌐 落尘之木 提供 123 网盘、夸克网盘、百度网盘。
🌐 QiuQuan 提供 官方链接、蓝奏云、123 网盘。
🌐 mefcl 提供 123 网盘、天翼云、夸克网盘

🌐 墨涩网 提供 官方链接、蓝奏云、123 网盘、内容较老。
🌐 易破解 提供 百度网盘、夸克网盘。

🌐 Softonic 提供软件的介绍和下载，类似于国内的“华军下载”，无破解内容。

🌐 FileCR 号称世界最大

🌐 iGet Into PC 大型软件，专业软件。
🌐 CrackingCity

🌐 Free4PC 博客网站，提供国外网盘下载，资源较老。
🌐 REPACK.ME 俄罗斯网站、
🌐 LRepacks 俄罗斯网站、
🌐 positiverecords.ru 俄罗斯网站、提供 BT 下载，以 Windows 主为。
🌐 monkrus.ws 俄罗斯网站、一个专门破解 Adobe 软件网站。
🌐 DIAKOV.NET 俄罗斯网站、但游客身份只提供慢速网盘下载。
▼ 特色软件
🌐 SourceForge GitHub 兴起前的一个软件托管平台。
🌐 PortableApps 一个专门提供各软件“便携版本”的网站。
🌐 Old versions 一个专门提供各软件“历史版本”的网站。
🌐 FileHippo.com 一个专门提供各软件“最新版本”的网站。
▼ 专业软件
🌐 奇迹秀工具箱 一个专门提供“设计类软件”的网站，Win Mac 都有。
🌐 软仓 收录了 391 个专业软件，需关注公众号，可订阅”去弹窗广告“过滤规则。
🌐 0DayDown 收录稀缺的 0day 软件，提供国外网盘下载，速度较慢，百度网盘要会员。
▼ 反馬软件
🌐 落尘之木 很多软件被”苏州思杰马克丁“公司代理而无法找到，这里是资源汇总。
🌐 脑袋瓜子 自建网盘下载，地址：瓜子云盘镜像
🌐 懒得勤快 自建网盘下载，地址：懒勤网盘

四、Mac 软件

▼ 国内网站 简介
🌐 MacApp分享频道 免注册、高速下载、无广告、资源首发。
🌐 MacYY 免注册、高速下载、无广告、资源丰富。
🌐 xClient 免注册、高速下载、有广告、资源丰富。
🌐 XMac.App 提供 国外网盘和 IPFS 下载，应用来自 TNT、HCiSO 等知名团队。
🌐 Digit77 开启广告屏蔽工具会隐藏下载链接。
🌐 Mac8k 提供 百度网盘下载
🌐 史蒂芬周 需要登陆
🌐 Mac萌新 需要等待 6 秒
🌐 Mac帮 提供 百度网盘下载，需要注册，回复评论才能下载。
🌐 知您网 提供 城通网盘下载。
🌐 优乐事 提供 城通网盘下载。
▼ 国外网站
🌐 MacTorrents 提供 BT 下载，TNT 和 HCISO 团队软件的源头。
🌐 AppStorrent 提供 国外网盘下载，俄罗斯知名 Mac 软件下载网站，这里也是源头。
🌐 Haxmac 提供 国外网盘下载。
🌐 Cmacked 提供 国外网盘下载，资源较老。
▼ 正版应用
🌐 MacUpdate
🌐 SetApp MacPaw 推出的订阅制付费 App 平台服务。
🌐 荔枝数码
🌐 Steam 很多 Mac 端游戏上不了 App Store，只能在 Steam 发布。
🌐 淘宝 在淘宝购买的价格，往往要比官方价格低很多，拼多多同理。

🌐 MSDN, ITellYou 纯净的系统下载网站，还有 Office，但没有 Win11。

🌐 TechBench 俄罗斯网站，提供菜单选择式的 Windows 镜像下载。
